# 1.ABEL「台灣區塊鍊活動清單」 說明

## 目的: 
提供一份完整並分類過的活動清單，可以視您本身的需求主動訂閱，輕鬆掌握台灣區塊鍊活動！  
  
(ABEL= Awesome Blockchain Event List)

# 2.ABEL創辦理念

想解決選擇區塊鍊活動時的幾個痛點：  
1. 活動訊息發布管道太多元 (FB/LINE/Wechat/網站..)  
2. 活動主題多樣化(鍊圈/幣圈/礦圈）
3. 活動符合目的性(開發/法規/人際交流/投資/徵才）

因此，想透過「簡單」、「一致」的線上電子日曆形式，來增進更多人（新韭菜們）對區塊練活動的掌握。
